# Documentation Assets

This directory contains supporting materials for CyberBRIEF documentation.

## Contents

Currently empty. This folder is reserved for:

- **Diagrams**: Architecture flowcharts, data flow visualizations
- **Screenshots**: UI examples, report samples
- **Examples**: Sample reports, MITRE Navigator exports, citation examples
- **Schema**: Database schema diagrams, API request/response samples
- **Guides**: Step-by-step tutorials, video thumbnails

## Future Assets

As documentation grows, add:

- `architecture-diagram.png` - System architecture visualization

- `sample-report.md` - Example BLUF report
- `mitre-navigator-export.json` - Sample ATT&CK mapping

## Contributing

When adding assets:
1. Use descriptive filenames
2. Optimize images (PNG: `pngquant --speed 1 file.png`)
3. Keep SVG diagrams clean and minimal
4. Include alt text in documentation links
5. Update this README with an entry
